# 濮阳便民电子地图开源网站
纯静态前端地图，基于Leaflet开发，无需后端、无需密钥，本地直接打开，支持GitHub Pages在线访问
覆盖区域：华龙区、濮阳县、清丰县、南乐县、范县、台前县

## 内置功能
1. 濮阳全域电子地图，拖拽、缩放
2. 六大分类点位标记：景区、医院、政务、公交、住宅小区
3. 地点搜索定位功能
4. 分专题独立页面，分类查看点位
5. 移动端自适应，手机电脑均可访问
6. 内置龙城新居等濮阳热门小区坐标

## 本地运行
1. 下载全部代码，解压文件夹
2. 双击 index.html 直接打开地图，无需服务器

## Github Pages部署
1. Github新建公开仓库，上传全部文件
2. 仓库设置 -> Pages
3. 分支选择main，文件夹选择根目录(root)
4. 保存等待2分钟生成在线网址

## 技术栈
Leaflet地图框架 + Bootstrap5 + HTML/CSS/原生JS
