# 濮阳市本地便民地图网站
纯静态开源地图平台，基于Leaflet开发，支持GitHub Pages一键部署
覆盖：华龙区、濮阳县、清丰县、南乐县、范县、台前县

## 功能
1. 濮阳市全域电子地图，缩放拖拽、定位
2. 六大区县边界文字标注
3. 旅游景点一键标记（戚城公园、龙湖、水秀、仓颉陵、单拐革命旧址等）
4. 全市医院、社区卫生服务中心分布
5. 政务大厅、派出所、车管所点位查询
6. 公交线路、公交站点地图展示
7. 小区名称检索、定位跳转
8. 测距、面积测量工具
9. 移动端自适应

## 部署方式
1. 新建GitHub仓库，上传全部代码
2. Settings -> Pages，分支选main，根目录部署
3. 自动生成在线地图链接

## 技术栈
Leaflet.js + HTML5 + Bootstrap5 + 原生JS
