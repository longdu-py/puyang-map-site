# 濮阳便民电子地图开源网站
纯静态前端地图，基于Leaflet开发，无需后端、无需密钥，本地直接双击index.html打开，支持GitHub Pages在线访问
覆盖区域：华龙区、濮阳县、清丰县、南乐县、范县、台前县

## 修复说明
修复本地打开地图空白、高度渲染失效、JS加载阻塞问题，自带地图加载提示。

## 内置功能
1. 濮阳全域电子地图，拖拽、滚轮缩放
2. 五大分类点位：景区、医院、政务大厅、公交、住宅小区（含龙城新居）
3. 搜索框一键定位地点，支持回车搜索
4. 独立专题分类页面
5. 手机/电脑自适应布局

## 本地运行步骤
1. 新建文件夹 puyang-map
2. 严格按照目录创建 assets/css、assets/js、pages 子文件夹
3. 复制全部代码对应粘贴，文件名不能修改
4. 双击根目录 index.html 立刻加载地图（需要联网加载底图）

## GitHub Pages部署
1. GitHub新建公开仓库，上传全部文件
2. Settings → Pages
3. 分支 main，目录选择 (root)
4. 保存等待1分钟生成在线网址

## 技术栈
Leaflet 1.9.4 + Bootstrap5 + HTML/CSS/原生JS
